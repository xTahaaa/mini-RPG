# Glitch-Arena
# 1. Cloner le dépôt
`git clone https://github.com/xTahaaa/mini-RPG.git`

`cd mini-RPG`

# 2. Créer le répertoire de build
`mkdir build && cd build`

# 3. Générer et compiler
`cmake ..`

`cmake --build .`

# 4. Exécution
  Sur Linux/macOS :

  `./mini-RPG`

  Sur Windows :
  
  `.\mini-RPG`
